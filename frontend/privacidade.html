Política de Privacidade Termos de Serviço Exclusão de Dados

1. Quem somos

O AutomacaoAds é uma plataforma de automação de anúncios para o Meta Ads (Facebook e Instagram). Esta política descreve como coletamos, usamos e protegemos suas informações ao utilizar nosso sistema.

2. Dados que coletamos

Para o funcionamento da plataforma, coletamos e armazenamos:

  • Credenciais da Meta API: App ID, App Secret e Access Token, necessários para criar e gerenciar anúncios em seu nome.
  • Ad Account ID: Identificador da sua conta de anúncios no Meta.
  • Dados de campanhas: Nomes, objetivos, orçamentos, métricas de performance (impressões, cliques, CTR, CPC).
  • Conteúdo gerado: Copies, prompts de imagem e segmentações de público criados pela IA ou manualmente.
  • Logs de atividade: Histórico de ações realizadas pelo sistema (criação, pausa, otimização de campanhas).
⚠ Importante: Não coletamos dados pessoais dos usuários finais dos anúncios (como nome, e-mail ou CPF de consumidores). Os dados tratados são exclusivamente dados operacionais da conta Meta do anunciante.

3. Como usamos os dados

Os dados coletados são utilizados exclusivamente para:

  • Autenticar e realizar chamadas à Meta Marketing API em seu nome.
  • Criar, pausar, ativar e otimizar campanhas conforme suas instruções.
  • Gerar relatórios de performance das campanhas.
  • Melhorar os modelos de IA para geração de copy e segmentação ao longo do tempo (dados agregados e anonimizados).
  • Registrar histórico de ações para auditoria e rastreabilidade.

Não compartilhamos, vendemos ou cedemos seus dados a terceiros, exceto quando necessário para o funcionamento técnico dos serviços integrados (Meta API, Firebase Firestore, OpenAI).

4. Serviços terceiros integrados

  • Meta for Developers (Meta Platforms, Inc.): As campanhas e dados de anúncios são gerenciados via Meta Marketing API. Consulte a Política de Privacidade da Meta.
  • Firebase Firestore (Google LLC): Os dados operacionais são armazenados no Google Cloud Firestore. Consulte a Política de Privacidade do Google.
  • OpenAI: Os prompts de contexto do produto são enviados à API da OpenAI para geração de copy e imagens. Consulte a Política de Privacidade da OpenAI.

5. Armazenamento e segurança

Todas as credenciais são armazenadas no Google Firebase Firestore com acesso restrito por autenticação de service account. Nenhum dado é armazenado em servidores locais sem controle de acesso.

O acesso à plataforma é protegido por chave de API (X-API-Key). Em produção, recomendamos o uso de Google Secret Manager para armazenamento de tokens sensíveis.

Os Access Tokens da Meta têm validade limitada e devem ser renovados pelo próprio usuário. Não armazenamos senhas de contas Meta.

6. Retenção de dados

Os dados são mantidos enquanto a automação estiver ativa na plataforma. Logs de auditoria são mantidos em janela deslizante de até 100 entradas por conta. Históricos de analytics são mantidos por tempo indeterminado para fins de melhoria contínua.

Você pode solicitar a exclusão completa dos seus dados a qualquer momento através da nossa página de exclusão de dados.

7. Seus direitos

Em conformidade com a LGPD (Lei Geral de Proteção de Dados — Lei nº 13.709/2018), você tem direito a:

  • Acessar os dados que temos sobre sua conta.
  • Corrigir dados incorretos ou desatualizados.
  • Solicitar a exclusão de todos os seus dados.
  • Revogar o consentimento para tratamento dos dados a qualquer momento.

Para exercer qualquer um desses direitos, utilize a página de exclusão de dados ou entre em contato pelo e-mail abaixo.
